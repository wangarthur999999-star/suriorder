# 苏里南本土洞察

## 市场特征

### 1. 关系社会
- "不认识的人，你门都进不去"
- 商业决策靠熟人推荐，不靠广告
- 族群内部信息传递快（华人跟华人讲、印度裔跟印度裔讲）
- **这是壁垒，不是劣势。** 外国公司撞墙更惨

### 2. 现金社会
- 大量现金交易
- 线上支付不是必须（COD 够用）
- 存在赊账/月结文化（熟人之间）
- **机会：** 系统做"挂账"功能，老板跟熟客月底结

### 3. 数字化程度
- 餐厅普遍不用 SaaS（甚至不用电脑）
- 很多小餐厅没有账本
- 老板用 WhatsApp 但不用邮箱
- 顾客手机存储小，不愿下载 App
- **所以：** WhatsApp-first + 原生 HTML（极轻量）是对的

### 4. 多语言
- 荷兰语（官方）、英语、中文、西班牙语、Sranan Tongo（口语）
- 四语不是加分项，是准入证
- 不做多语言就有一群人用不了

### 5. 基础设施
- 偶尔停电断网
- 极轻量技术选型在这个环境下反而是优势
- 未来考虑离线模式

---

## 华人商圈特殊性

- 老板之间消息传得快
- 一个用了说好 → 整个圈子很快知道
- 一个用了说不好 → 你也很快出局
- **第一家店的口碑是零和一的关系，不是及格分**
