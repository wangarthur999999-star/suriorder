# 竞争分析

## 现有竞争对手

苏里南已有外卖公司。分析差异：

| 维度 | 外卖公司 | SuriOrder |
|------|---------|-----------|
| 配送 | 自建骑手，重资产 | 无配送，到店取 |
| 抽成 | 20-30% | 0%（SaaS 免费） |
| 目标 | 替代餐厅流量 | 帮餐厅做增量 |
| 关系 | 餐厅 vs 平台（博弈） | 餐厅 vs 数据（合作） |
| 立场 | "我是平台，你在我这卖" | "我是你的人，帮你多赚钱" |

## 壁垒

### 1. 餐厅关系（最强壁垒）
苏里南靠族群信任。华人圈子你进去了，外来者除非也认识华人老板，否则进不来。这是**文化壁垒**，不是技术壁垒。

### 2. 菜单数据（累积壁垒）
苏里南没有 Zomato/美团。你积累的就是唯一的。每多一家店，数据库就更难被替代。**先发优势随时间增强。**

### 3. 顾客习惯（迁移成本）
顾客 WhatsApp 聊天记录里有你的链接。下次再点，打开聊天记录就行。**链接留在聊天记录里 = 天然复购路径。**

### 4. B 端合同（合约壁垒）
企业团餐合同一旦建立，竞争对手要撬，成本很高。**B 端客户的粘性远超 C 端。**

### 5. 本土认知（信息壁垒）
你在苏里南生活，了解族群结构、消费习惯、价格敏感度。外国公司做市场调研都摸不清。

## 别人抄你的难度

| 能抄的 | 难抄的 |
|--------|--------|
| 代码（同类软件很多） | 餐厅关系（需要人脉和时间） |
| UI/UX | 菜单数据（积累需要时间） |
| 技术栈 | 族群信任（你不是华人很难进华人圈） |

**结论：抄代码容易，抄信任难。** 你的壁垒不是技术，是关系和数据的复利效应。

## 风险矩阵

| 风险 | 概率 | 应对 |
|------|------|------|
| 餐厅不改变习惯 | 高 | 不卖改变，卖增量收入 |
| Render 免费层休眠 | 高 | B 端有量后上 $7/月 Starter |
| 有人抄你 | 中 | 关系+数据壁垒；抄代码没用 |
| WhatsApp API 政策变 | 低 | 免费层够用很久 |
| 餐厅流失 | 中 | 做了 B 端团餐后，离开你=失去 B 端订单，流失成本变高 |
